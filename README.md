## Features

1. Sign up/resiter user
2. Auto authorisation with encrypted token
3. Adding cards
4. Editing user data
5. An opportunity to delete only yours cards
6. Showing zommed pictures by clicking on cards

## Install && Launch

1. check MongoDB path in backend/app.js:23

2. Frontend

```
cd frontend/
yarn install
yarn start
```
Backend

```
cd backend/
yarn install
yarn dev
```
